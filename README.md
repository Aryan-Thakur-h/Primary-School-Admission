# MinorProjectTT - KidKinder Kindergarten Website

## Description
MinorProjectTT is a static website template designed for a kindergarten or kids' learning center called KidKinder. It provides an engaging and informative platform to showcase educational services, classes, teachers, and facilities for young children. The website is built using HTML, CSS, and JavaScript, featuring responsive design and interactive elements.

## Features
- **Home Section**: Introduction to the learning center with a call-to-action.
- **Facilities**: Highlights key amenities like playground, music and dance, arts and crafts, safe transportation, healthy food, and educational tours.
- **About Us**: Information about the school and its benefits.
- **Classes**: Details on popular classes including Drawing, Language Learning, and Basic Science, with age groups, seats, timings, and fees.
- **Teachers**: Profiles of the teaching staff with photos and social links.
- **Blog**: Latest articles and educational resources.
- **Contact/Registration**: Form for booking seats and contact information.
- **Responsive Design**: Optimized for various devices using Bootstrap.
- **Interactive Elements**: Carousel, lightbox, and smooth scrolling.

## Technologies Used
- HTML5
- CSS3 (with Bootstrap framework)
- JavaScript (jQuery, Owl Carousel, Lightbox)
- Font Awesome for icons
- Google Fonts for typography

## Installation and Setup
1. Clone or download the project files to your local machine.
2. Ensure all files are in the `MinorProjectTT` directory.
3. Open `index.html` in a web browser to view the website.
4. No server setup is required as it's a static site.

## Usage
- Navigate through the sections using the navbar.
- View class details and join classes via the provided buttons.
- Contact the school using the footer information or registration form.
- Explore the blog for educational content.
